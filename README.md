## Quick Setup

``` bash
$ virtualenv env

# Activate venv
$ source env/bin/activate

# Install requirements
$ pip3 install -r requirements.txt

# Migrations
$ python manage.py makemigrations
$ python manage.py migrate

# Run Server (http://localhost:8000)
python3 manage.py runserver
```
